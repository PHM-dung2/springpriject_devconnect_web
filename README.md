## 📌 주요 기능
- **개발자와 회사가 서로를 평가**
  - 평가 및 점수를 추가함으로써 프로젝트 선택 및 판단 가능
  

    
- **프로젝트 검색, 탐색, 매칭 추천**
  - 프로젝트 검색(백엔드, 프론트)
  - 프로젝트 탐색 
  - 기술스택을 매칭


    
- **react 관리자 페이지에서 프로젝트 상태 확인**
  - 모니터링 및 조정.
  - 
  -
 
- **회사와 개발자 각 로그인 기능 및 개발자 레벨과 랭킹 시각화**
  - 개발자 레벨과 랭킹을 통해 성장 동기부여
  - 기업의 고용시 직관적인 평가 수치 제공
